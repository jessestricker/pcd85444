# pcd8544
A C++ shared library for easy control of the PCD 8544 from the Raspberry Pi

## Dependencies

* C++ 11
* [bcm2835](http://www.airspayce.com/mikem/bcm2835)
* [gpio](https://github.com/jessestricker/gpio)

## Building

```
make
sudo make install
```

## Testing

```
make test
sudo ./gpio-test
```
